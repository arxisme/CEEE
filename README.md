# Machine Learning Lab Course

Hey everyone, welcome to the Machine Learning Lab Course! I'm super excited to guide you through this journey. Over the next four sessions, we're going to get our hands dirty with real code, starting from basic environment setup all the way to using AI coding agents to write and grade our models. 

Each session we have planned out will take us about an hour and a half to two hours to get through, so we’ll keep things focused and practical. 

## What We'll Be Doing

*   **[Day 1: Getting Set Up](Day-1/)**: Before we can build anything, we need the right tools. We'll set up your laptops with Python and Conda, and I'll show you Google Colab for those who want to run things in the cloud.
*   **[Day 2: Traditional Models](Day-2/)**: We're going to dive into classic algorithms using `scikit-learn`. We'll write some code to classify data and actually visualize how these models make decisions.
*   **[Day 3: Deep Learning & HuggingFace](Day-3/)**: This is where things get really cool. We'll jump onto Colab to grab some free GPUs, learn how text tokenization works, and use pre-trained models from HuggingFace.
*   **[Day 4: AI Tools & Autograding](Day-4/)**: For our last session, I’m going to show you how software engineers actually work today using AI assistants to debug. We'll also build an autograder to evaluate our models automatically!
