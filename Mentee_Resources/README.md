# Machine Learning Lab Course: Reference Guide

This repository contains the required resources, code, and conceptual documentation for the Machine Learning Lab Course. It serves as a self-revision guide for attendees.

## Course Structure

The curriculum is divided into four distinct sessions:

*   **[Day 1: Environment Configuration](Day-1/)**: Details the process for establishing a functional ML environment both locally (via Python virtual environments) and in the cloud (via Google Colab).
*   **[Day 2: Traditional Machine Learning](Day-2/)**: Covers the fundamental pipeline for training classification models using `scikit-learn`, alongside decision boundary visualization techniques.
*   **[Day 3: Deep Learning and NLP](Day-3/)**: Introduces Transformer architectures, tokenization mechanics, and cloud-based GPU allocation using the HuggingFace ecosystem.
*   **[Day 4: Automated Evaluation and AI Tooling](Day-4/)**: Explores modern software engineering practices, specifically the use of AI assistants for code debugging and the development of automated model evaluation pipelines.

Each directory contains a detailed `README.md` documenting the conceptual theory and execution steps for the provided scripts and notebooks.
